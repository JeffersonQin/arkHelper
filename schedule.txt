#暂未实装！
# 作战计划以START开始，END结束
# 本文件大小写不敏感，即th与TH是一样的
# 一个正确的关卡代号格式应当是：a-b-c:times
# :为英文中的冒号
# a为模块，对应关系为：
# 主线：TH
# 物资筹备：RS
# 芯片搜索：PR
# 剿灭作战：EX
# b为章节，对应关系为：
# 主线关卡与章节号一致，如第6章即为6，第5章S关卡即为S5
# 物资筹备与游戏中代号一致，如作战记录即为LS
# 芯片搜索与游戏中代号一致，如重装治疗芯片为A
# 剿灭作战从左至右排序，如切尔诺伯格为01，龙门市区为03
# c为关卡号，对应关系与游戏中一致
# 剿灭作战c部分填写为00
# times部分为重复该关卡次数，不可省略
# 如填写为-1，则将重复该关卡到理智耗尽
# 下为示例，自己配置时请删除
START
TH-1-7:2
RS-LS-5:1
END
# 意为打1-7两次，LS-5一次
